## 📖 About the Project

**Net Worth Tracker** is a private, web-based analytics engine designed to help users understand their assets, liabilities, and long-term financial trajectory with mathematical precision. 

Moving beyond standard budgeting apps, this platform focuses on high-level wealth tracking. It allows users to log point-in-time financial snapshots, categorize complex asset portfolios, and visualize historical trends through a clean, distraction-free interface.

### ✨ Core Features
* **Dynamic Ledger:** Categorize and track diverse assets and liabilities with granular, spreadsheet-level control and rapid inline editing.
* **Immutable Snapshots:** Log point-in-time financial data to build an accurate, unalterable history of your wealth accumulation.
* **Trend Analytics:** Visualize portfolio allocation, track top financial movers, and map your net worth trajectory over time using composed charts.
* **Financial Goals:** Set, track, and automatically link custom financial milestones to your real-time net worth.
* **Secure Authentication:** Frictionless Google OAuth integration with a strict zero-access data policy.

---

## 🔒 Security & Privacy Philosophy

Financial data requires absolute trust. This application is engineered with a **Private by Design** methodology:
1. **Zero-Access Policy:** Authentication is handled entirely by Google's OAuth servers. The application never sees or stores passwords.
2. **Encrypted Storage:** All ledger entries and historical snapshots are securely transmitted via HTTPS and encrypted at rest on Google Cloud infrastructure.
3. **No Monetization:** User data is never monitored, analyzed for marketing, sold, or shared with third parties.
